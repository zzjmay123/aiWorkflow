**适合有技术基础的用户**

- ✅ Python 3.8+ 环境
- ✅ 已安装 Git
- ✅ 小红书账号已实名认证

#### 安装命令

```bash

cd ai-workflow-skill

# 2. 运行安装脚本（自动安装依赖）
```
