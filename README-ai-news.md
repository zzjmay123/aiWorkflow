## 🛠️ 技术细节

### 浏览器自动化流程

```python
1. 启动 Chromium（无头模式）
2. 打开 ai-news-generator.html
3. 填写表单：
   - 主标题：AI 资讯快报
   - 日期：2026.04.04
   - 正文：粘贴 10 条资讯
4. 等待自动解析（0.5 秒）
5. 检查状态（绿色=成功）
6. 逐页下载：
   - 点击"下一页"
   - 点击"下载当前页"
   - 保存图片
7. 关闭浏览器
8. 发送飞书
```

### 依赖库

```bash
pip3 install playwright
playwright install chromium
```
