# 安装指南

> 本技能支持 **CoPaw** 和 **OpenClaw** 两种平台，任选其一即可。

---

## 方式一：CoPaw 安装（推荐）

### 1. 使用 CoPaw 命令安装

```bash
copaw skill install git@github.com:zzjmay123/aiWorkflow.git --branch ai-workflow-skill
```

### 2. 进入技能目录

```bash
cd ~/.copaw/skills/ai-workflow
```

### 3. 运行安装脚本

```bash
bash install.sh
```

### 4. 授权小红书

```bash
xiaohongshu auth login
```

---

## 方式三：手动克隆（通用）

适合不使用 CoPaw/OpenClaw 的独立环境。

### 1. 克隆仓库

```bash
git clone -b ai-workflow-skill git@github.com:zzjmay123/aiWorkflow.git ai-workflow-skill
cd ai-workflow-skill
```

### 2. 运行安装脚本

```bash

```

### 3. 授权小红书

```bash
xiaohongshu auth login
```

## 环境要求

### Python 版本

需要 **Python 3.8+**

```bash
# 检查版本
python3 --version
```

### 各系统安装 Python

**macOS**
```bash
brew install python@3.10
```

**Windows**
前往 https://www.python.org/downloads/ 下载 Python 3.10+，安装时勾选 "Add to PATH"

**Linux (Ubuntu/Debian)**
```bash
sudo apt update
sudo apt install python3 python3-pip
```

## 安装后配置

### 1. 安装依赖

```bash
cd ai-workflow-skill
pip install -r requirements.txt

# 安装浏览器（用于截图）
playwright install chromium
```

### 2. 安装小红书 CLI

```bash
# 安装 pipx
pip install pipx
pipx ensurepath

# 安装小红书官方 CLI
pipx install xiaohongshu-cli

# 登录授权（需扫码）
xiaohongshu auth login
```

**注意**：首次登录需要在小红书 App 内扫码确认，完成账号绑定。

### 3. 验证安装

```bash
# 检查小红书 CLI
xiaohongshu --version

# 测试运行
cd scripts
python auto_publish_final.py
```

## 图片生成器使用说明

1. 用浏览器打开 `tools/ai-news-generator.html`
2. 修改标题和日期
3. 粘贴生成的文本内容（来自 `ai_news_text_only.py`）
4. 点击"下载图片"按钮
5. 图片将保存到下载文件夹

## 常见问题

### 安装报错？
检查 Python 版本是否 ≥3.8

### playwright 安装失败？
尝试 `playwright install-deps`

### 小红书登录失败？
确保小红书 App 是最新版本

### 找不到命令？
- CoPaw: 确认 `copaw` 在 PATH 中
- OpenClaw: 确认 `claw` 在 PATH 中
