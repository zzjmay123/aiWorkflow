# 安装指南

## 方式一：CoPaw 安装（推荐）

### 1. 使用 CoPaw 命令安装

```bash
copaw skill install git@github.com:zzjmay123/aiWorkflow.git --branch ai-workflow-skill
```

### 2. 进入技能目录

```bash
cd ~/.copaw/skills/ai-workflow
```

### 3. 运行安装脚本

```bash
bash install.sh
```

### 4. 授权小红书

```bash
xiaohongshu auth login
```

## Python 环境准备

### macOS
```bash
# 检查 Python 版本（需要 3.8+）
python3 --version

# 如果未安装，使用 Homebrew 安装
brew install python@3.10
```

### Windows
前往 https://www.python.org/downloads/ 下载 Python 3.10+，安装时勾选 "Add to PATH"

### Linux (Ubuntu/Debian)
```bash
sudo apt update
sudo apt install python3 python3-pip
```

## 2. 安装依赖

```bash
cd ai-workflow-sell
pip install -r requirements.txt

# 安装浏览器（用于截图）
playwright install chromium
```

## 3. 安装小红书 CLI

```bash
# 安装 pipx（Python 包管理工具）
pip install pipx
pipx ensurepath

# 安装小红书官方 CLI
pipx install xiaohongshu-cli

# 登录授权（需扫码）
xiaohongshu auth login
```

**注意**：首次登录需要在小红书 App 内扫码确认，完成账号绑定。

## 4. 验证安装

```bash
# 检查小红书 CLI
xiaohongshu --version

# 测试发布（会生成测试笔记）
cd scripts
python auto_publish_final.py
```

## 5. 图片生成器使用说明

1. 用浏览器打开 `tools/ai-news-generator.html`
2. 修改标题和日期
3. 粘贴生成的文本内容（来自 ai_news_text_only.py）
4. 点击"下载图片"按钮
5. 图片将保存到下载文件夹
