# 常见问题 (FAQ)

## 安装问题

### Q: Python 版本报错？
**A**: 系统需要 Python 3.8 或以上版本。运行 `python3 --version` 检查，过低请升级。

### Q: playwright 安装失败？
**A**: 
1. 尝试 `playwright install-deps` 安装系统依赖
2. macOS 用户可能需要 `brew install` 相关库
3. 防火墙可能拦截，尝试关闭后重试

### Q: xiaohongshu-cli 无法登录？
**A**:
1. 确保小红书 App 是最新版本
2. 扫码后 2 分钟内完成确认
3. 网络环境需能访问小红书

## 运行问题

### Q: 新闻抓取不到？
**A**:
1. 检查网络连接
2. 网站可能改版，需更新 `auto_news_browser.py` 中的选择器
3. 部分媒体有反爬机制，可尝试降低抓取频率

### Q: 图片生成空白？
**A**:
1. 确保粘贴的内容格式正确（参考输出文件）
2. 浏览器需支持 HTML5（Chrome/Edge 推荐）
3. 检查 `ai-news-generator.html` 是否完整

### Q: 发布失败？
**A**:
1. 确认 `xiaohongshu auth login` 已成功
2. 图片格式/大小不符合要求
3. 文案包含敏感词
4. 账号被限制发布

## 配置问题

### Q: 飞书通知收不到？
**A**:
1. 检查 App ID 和 Secret 是否正确
2. 确认应用权限包含 `im:message`
3. 接收人 ID 是否正确（非手机号）

### Q: 想改成其他领域（如财经/体育）？
**A**:
1. 修改 `auto_news_browser.py` 中的 `sources` 列表
2. 更改 AI 关键词过滤逻辑
3. 调整图片标题和标签

### Q: 每天发布次数限制？
**A**: 系统内置防重复机制（锁文件），每天一次。如需多次，删除 `/tmp/xiaohongshu_publish_lock` 文件。

## 优化建议

1. **定期维护**：网站改版后需更新选择器
2. **账号安全**：建议用副号测试，稳定后再用主号
3. **内容审核**：AI 生成的内容建议人工审核后发布
4. **备份数据**：定期备份脚本和配置
