# 配置说明

## 1. 脚本配置

打开 `scripts/ai_news_text_only.py`，修改以下参数：

```python
# 飞书配置（用于发送通知）
FEISHU_APP_ID = "你的飞书应用 ID"
FEISHU_APP_SECRET = "你的飞书应用密钥"
RECEIVE_USER_ID = "接收人飞书 ID"

# 小红书发布配置
XHS_MODE = "public"  # public=公开，private=私密
```

## 2. 飞书应用配置

1. 前往 https://open.feishu.cn/ 创建应用
2. 获取 App ID 和 App Secret
3. 添加权限：`im:message`、`im:message:send_as_bot`
4. 发布应用，获取接收人 ID（可通过机器人发送消息查看）

## 3. 图片生成器配置

打开 `tools/ai-news-generator.html`，可修改：

```javascript
// 图片尺寸（默认 3:4）
const width = 540;
const height = 720;

// 每页显示条数（默认 2 条）
const newsPerPage = 2;

// 颜色主题
const borderColor = '#000';  // 边框颜色
const titleColor = '#000';   // 标题颜色
```

## 4. 新闻源配置

打开 `scripts/auto_news_browser.py`，修改 `sources` 列表：

```python
sources = [
    {"name": "量子位", "url": "https://www.qbitai.com/", ...},
    {"name": "36 氪", "url": "https://36kr.com/", ...},
    # 可添加/删除/修改新闻源
]
```

## 5. 定时任务配置

### macOS/Linux (cron)
```bash
crontab -e

# 每天早 8 点
0 8 * * * cd /path/to/ai-workflow-sell/scripts && python ai_news_text_only.py

# 每天晚 8 点
0 20 * * * cd /path/to/ai-workflow-sell/scripts && python auto_publish_final.py
```

### Windows (任务计划程序)
1. 打开"任务计划程序"
2. 创建基本任务
3. 设置触发时间和执行程序

## 6. 防重复发布

系统使用锁文件机制，每天只发布一次：
```
/tmp/xiaohongshu_publish_lock  # macOS/Linux
C:\Temp\xiaohongshu_publish_lock  # Windows
```

如需重新发布，删除该文件即可。
