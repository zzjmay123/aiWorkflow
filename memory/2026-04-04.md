#### 1. 图片样式不一致问题
**问题**: Python PIL 生成的图片与 HTML 预览效果不一致
- PIL 生成：简单黑白样式，字体默认
- HTML 生成：小红书风格，精美圆角卡片

**解决**: 
- 采用浏览器自动化方案（auto_news_browser.py）
- 使用 Playwright 操作 HTML 生成器
- 确保生成的图片和用户预览完全一致

**问题**: 测试时所有新闻来源都写成了"量子位"
**原因**: 为了快速验证，临时硬编码了单一来源

- 来源分布：量子位 4 条、机器之心 3 条、36 氪 1 条、TechCrunch 1 条、AI 前沿观察 1 条

**洞察**: 用户上班途中无法使用电脑
